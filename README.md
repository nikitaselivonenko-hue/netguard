\# NetGuard – Lightweight Network Intrusion Detection System



Система обнаружения вторжений (IDS) с анализом сетевого трафика, выявлением аномалий и оповещением в Telegram.



\## Возможности



\- \*\*Обнаружение атак:\*\*

&#x20; - ARP-спуфинг (отслеживание смены MAC-адреса)

&#x20; - Сканирование портов (SYN-сканирование)

&#x20; - DNS-туннелирование (длинные домены, высокая энтропия, TXT-записи, высокая частота запросов)

&#x20; - ICMP-туннелирование (большие пакеты, base64 в payload, высокая частота)

\- \*\*Загрузка PCAP-файлов\*\* – офлайн-анализ трафика без необходимости живого захвата.

\- \*\*Генерация демонстрационных алертов\*\* – позволяет сразу увидеть интерфейс в действии.

\- \*\*Уведомления в Telegram\*\* – алерты дублируются в мессенджер.

\- Веб-интерфейс на Flask с отображением алертов в реальном времени.



\## Технологии



\- Python 3, Flask, Scapy

\- SQLite (хранение алертов)

\- Bootstrap 5 (интерфейс)

\- Telegram Bot API
